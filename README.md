# telecom-analysis

## Objetivo del proyecto.
- analizar el negocio usando los 3 datasets (clientes, uso_real y planes) para que ConnectaTel entienda cómo se comportan sus usuarios según edad, volumen de llamadas/mensajes y nivel de consumo.

## datasets utilizados:
- ver carpeta data(users,usage,plans)

## etapas del análisis realizadas
- Cargar y explorar
- Identificación de problemas de calidad
- Limpieza básica
- Summary statistics
- Visualización & outliers
- Segmentación
- Insight ejecutivo

## Como ejecutar el notebook
- Abrir el notebook en Google Colab o Jupyter Notebook.
- Ejecutar las celdas en orden.
- Revisar los resultados y visualizaciones generadas

## Reproducibilidad
- Proyecto puede reproducirse utilizando los datasets disponibles en la carpeta `data`
